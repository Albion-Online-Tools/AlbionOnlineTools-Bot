## ‚öîÔ∏è Albion Online Bot Commands

| Command           | Description                                                                |
|-------------------|----------------------------------------------------------------------------|
| `/price`          | Check buy/sell orders and price history graph for any item.                 |
| `/player`         | View detailed stats about any player.                                      |
| `/guild`          | View detailed stats about any guild.                                       |
| `/gold`           | Live gold price and historical chart.                                      |
| `/randomator`     | Get a random Albion build (including spells).                              |
| `/image`          | High quality images of any item with all qualities.                        |
| `/server-status`  | Live server status feed for Albion Online.                                 |
| `/builds build`   | Display a specific build.                                                  |
| `/builds new-build` | Create a new Albion build.                                              |
| `/builds edit-build`| Edit an existing build.                                                 |
| `/builds remove-build`| Remove a build.                                                       |
| `/builds set-builder-role`| Set a role to manage builds on your server.                        |
| `/killboard info` | Show killboard information for this server.                                |
| `/killboard set-channel`| Set channel for kill/death feed.                                     |
| `/killboard track`| Track kills/deaths for players or guilds.                                  |
| `/killboard untrack`| Remove specific trackers.                                                |
| `/killboard remove`| Reset the killboard for this server.                                      |
